About Web Share API

The Web Share API allows websites to share text, links, and files to native sharing capabilities of the device.

This demo shows how to implement the Web Share API in modern browsers. Note that this API is only available in secure contexts (HTTPS) and on supported browsers.

Browser Support:

  • Chrome: ✔️ (Android & Desktop)
  • Safari: ✔️ (iOS & macOS)
  • Edge: ✔️ (Android & Desktop)
  • Firefox: ❌ (Not supported)
